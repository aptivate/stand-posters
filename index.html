What's in here

Scripts to build a browser kiosk and virtual images with offline versions of some of Aptivate's applications. The later is work in progress as of writing.

Installing

Stand-posters' kiosk requires Ubuntu Raring 13.04.

To install type the following in your terminal. Requires sudo privileges to install packages in your system

wget -qO- https://github.com/aptivate/stand-posters/raw/master/bin/install-posters | bash

Once installed the kiosk can be launched typing 'AptivateDemo' in Unity.
